## Run Locally

Clone the project

```bash
  git clone https://github.com/azariaberyl/tiimz.git
```

Go to the project directory

```bash
  cd tiimz
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
